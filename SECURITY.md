# Security Policy

## Reporting a vulnerability

Please report security vulnerabilities **privately** — do not open a public issue.

Email **mazher@gmail.com** with a description of the issue and steps to reproduce. You can
expect an initial response within a few days. Once the issue is confirmed and fixed, the fix
will be released and the report acknowledged (with credit, if you'd like).

## Supported versions

Parchmint is pre-1.0; only the latest released version is supported with security fixes.
